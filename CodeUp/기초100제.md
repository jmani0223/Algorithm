### 6. 출력하기06
퍼센트 문자 %를 출력하기 위해서는 %%로 출력해 주어야 한다.
```c
#include <iostream>
using namespace std;

int main() {
	printf("\"!@#$%%^&*()\"");
	return 0;
}

/*
	"!@#$%%^&*()"
*/
```
	
### 7. 출력하기07

```c

	printf("\"C:\\Download\\hello.cpp\"");

*/
```

### 19. 연월일 입력받아 그대로 출력하기
%02d를 사용하면 2칸을 사용해 출력하는데, 한 자리 수인 경우 앞에 0을 붙여 출력한다.
```c
/* 
	2013.8.5
*/

#include <iostream>
using namespace std;

int main() {
	int y, m, d;
	scanf("%d.%d.%d", &y, &m, &d);
	printf("%04d.%02d.%02d", y, m, d);
	return 0;
}

/*
	2013.08.05
*/
```

### 20. 주민번호 입력받아 형태 바꿔 출력하기
앞의 6자리는 생년월일(yymmdd)이고 뒤 7자리는 성별, 지역, 오류검출코드이다.
```c
/* 
	000907-1121112
*/

#include <iostream>
using namespace std;

int main() {
	int y, m, d, e;
	scanf("%02d%02d%02d-%07d", &y, &m, &d, &e);
	printf("%02d%02d%02d%07d", y, m, d, e);
	return 0;
}


/*
	0009071121112
*/
```

### 22. 문장 1개 입력받아 그대로 출력하기
fgets( ) 를 사용하면 공백문자가 포함되어잇는 문장을 입력받아 저장할 수 있다.
만약, scanf("%s", ... ) 를 이용해 문장을 입력받으면, 첫 번째 단어까지만 저장된다.

```c
/* 
	Programming is very fun!!
*/

#include <iostream>
using namespace std;

int main() {
	char c[2001];
	fgets(c, 2000, stdin);
	printf("%s", c);
	return 0;
}


/*
	Programming is very fun!!
*/

```
