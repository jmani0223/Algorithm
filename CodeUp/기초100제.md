### 25. 정수 1개 입력받아 나누어 출력하기
문제를 응용하여 어느 자리든지 출력할 수 있도록 했다. 
다만, 가장 높은 자리부터 출력하기 위한 알고리즘이 아니기에 스택 자료구조를 이용했다. 
```c
/* 
	75254
*/

#include <iostream>
#include <string>
#include <stack>
using namespace std;

int main() {
	int num, trs;
	int k = 10, i;
	stack<int> s;
	
	cin >> num;
	
	for(i=1;num >10 ; i*=10 ){
		trs = num%k;
		num = num/k;
		s.push(trs*i);
	}
	s.push(num*i);
	
	for(int j=0; s.size(); j++){
		cout << "["<<s.top()<<"]"<<endl;
		s.pop();
	}
	return 0;                       
}


/*
	[70000]
	[5000]
	[200]
	[50]
	[4]
*/
```

### 47. 정수 1개 입력받아 2배 곱해 출력하기
왼쪽 비트시프트(<<)가 될 때에는 오른쪽에 0이 주어진 개수만큼 추가되고,
오른쪽 비트시프트(>>)가 될 때에는 왼쪽에 0(0 또는 양의 정수인 경우)이나 1(음의 정수인 경우)이 개수만큼 추가된다.
```c
/* 
	1024
*/

#include <iostream>
using namespace std;

int main() {
	int a;
	cin >> a;
	cout << (a<<1);
	return 0;
}

/*
	2048
*/
```
